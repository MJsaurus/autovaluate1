# AutoValuate

Marketing site for AutoValuate — independent, data-driven **diminished value** and **total loss** vehicle valuation reports, reviewed and signed by a licensed Texas adjuster.

## Structure

- `index.html` — the landing page (static, self-contained).

## Deploy

Hosted on Cloudflare Pages. Pushing to the connected GitHub repo triggers an automatic deploy.

## Roadmap

- Free-estimate calculator
